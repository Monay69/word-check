# 单词检测 · Word Check

一个**零依赖、纯前端、单文件**的背单词小工具。录入单词和释义，反复检测，**答错的会进入下一轮继续考，循环到全部掌握为止**。针对手机浏览器（尤其 iPhone Safari）优化，打开即用，数据全部留在你自己的浏览器里，不联网、不上传。

> 在线体验：https://monay69.github.io/word-check/

## 功能

- **多词库**：可建立任意多个词库，分门别类存放不同的单词集（如四级、考研、西语…），各自独立练习。
- **内置词书**：自带英语四级 / 六级高频精选、西班牙语入门基础，一键添加即可开练（精选高频版，非完整考纲）。
- **检测模式**：看词选义 / 看义选词 / 混合出题；答错自动归入下一轮，每轮只考没掌握的词，循环到全对通关。
- **配对模式**：Quizlet 风格的点击消除游戏，带计时，错配的词可单独再练。
- **掌握点阵**：顶部实时显示每个词的状态（当前 / 答对 / 答错），进度一目了然。
- **批量录入**：每行一个，单词和意思之间用空格 / Tab / 逗号 / 破折号都能识别，释义里带逗号也不会切错。
- **本地保存**：词库自动存在浏览器本地（localStorage），下次打开还在；不联网、不上传。
- **可安装 / 离线**：支持 PWA，iPhone 用 Safari「添加到主屏幕」后可全屏运行、有独立图标，并能离线使用。

## 本地使用

直接下载 `index.html`，用任意浏览器打开即可。无需安装、无需服务器。

## 部署到网上（GitHub Pages）

1. 在 GitHub 新建一个 **public** 仓库，例如命名 `word-check`。
2. 把本项目的文件（至少是 `index.html`）上传进去。
3. 进入仓库 **Settings → Pages**，在 *Source* 选 **Deploy from a branch**，分支选 `main`、目录选 `/ (root)`，保存。
4. 等一两分钟，访问 `https://<你的用户名>.github.io/word-check/` 即可。

> 入口文件必须叫 `index.html`，GitHub Pages 才会在根路径自动打开它。

## 技术说明

- 单个 `index.html`，原生 HTML / CSS / JavaScript，无任何构建步骤和第三方依赖。
- 数据通过浏览器 `localStorage` 持久化，仅存在用户本机。
- 适配移动端触控与安全区域，支持「添加到主屏幕」当作 App 使用。

## 隐私

所有词库数据只保存在使用者自己的浏览器中，应用本身不发起任何网络请求，不收集、不上传任何信息。
